Brewer's Cookbook

HTML - CSS - JavaScript - React

This project uses fetch API to get data from Punk API (Link). The user can retreive a list of random beers or by using the search they can filter for beers with a minimum ABV (alcohol by volume) content. React is used to manage the UI, beer list and individual beer recipe components, as well as all states. React routing is used to take the user from the beer list to the individual beer info. This is an ongoing project, in the future I plan to implement a backend as allow for users to add thier own beer recipes.
